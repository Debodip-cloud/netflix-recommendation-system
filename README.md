An end-to-end movie recommendation system built using the **Netflix Prize Dataset** and **Collaborative Filtering (Singular Value Decomposition - SVD)**.

The objective of this project is to recommend movies that a user is likely to enjoy by learning hidden relationships between users and movies from historical rating behaviour.

The project demonstrates the complete machine learning workflow, including:

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Collaborative Filtering
- Model evaluation
- Personalized movie recommendation
- Streamlit deployment

---

# Project Overview

Recommendation systems have become one of the most important applications of Machine Learning and Artificial Intelligence. Companies such as Netflix, Amazon and Spotify rely heavily on recommendation algorithms to personalize user experiences.

This project implements a collaborative filtering recommendation engine using the famous **Netflix Prize Dataset**, which contains millions of historical movie ratings from anonymous Netflix users.

Instead of recommending movies based on genres or manually created rules, the model learns latent relationships between users and movies using **Matrix Factorization (SVD)**.

## 3. Exploratory Data Analysis

The notebook performs several exploratory analyses including:

- Rating distribution
- Average rating
- Most popular movies
- Highest-rated movies
- User activity analysis
- Movie popularity analysis
- Rating statistics
- User-Movie Matrix Sparsity

Some interesting findings include:

- Average movie rating ≈ **3.60**
- Median rating = **4**
- Most ratings are either **3** or **4**
- User-Movie matrix sparsity ≈ **99.91%**

---

## 4. Data Preprocessing

To improve recommendation quality:

Users with fewer than **50 ratings** were removed.

Movies with fewer than **100 ratings** were removed.

The filtered dataset was then sampled for model training.

---

## 5. User-Movie Matrix

A User-Movie Rating Matrix was created where

Rows represent users

Columns represent movies

Values represent ratings

Missing values indicate that the user has not rated that movie.

---

## 6. Recommendation Model

The recommendation engine was developed using

### Singular Value Decomposition (SVD)

The model learns hidden latent factors representing

- User preferences
- Movie characteristics

These latent factors are then used to estimate ratings for movies a user has never watched.

---

# Model Evaluation

The trained recommendation model achieved

| Metric | Value |
|---------|-------|
| RMSE | **0.9842** |
| MAE | **0.7856** |

### Interpretation

An RMSE below **1** indicates that the recommendation model predicts user ratings with relatively small errors.

The MAE value indicates that predicted ratings differ from actual ratings by less than one rating point on average.
